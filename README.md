# 📝 TeraExam - Modern Exam Portal

TeraExam ek high-performance aur scalable online examination portal hai, jise modern web technologies aur serverless database ka use karke banaya gaya hai. Yeh application ek sath hazaaron students ke exam load ko smoothly handle karne ke liye design ki gayi hai.

---

## 🚀 Features

- **Neon Serverless Database Integration:** High availability aur automatic scaling ke sath fast data retrieval.
- **Modern UI/UX:** Responsive design jo mobile, tablet aur desktop sabhi par perfect chalta hai.
- **Optimized Performance:** Fast rendering aur low latency backend management.
- **Secure Architecture:** Sensitive credentials aur API keys completely protected hain.

---

## 🛠️ Tech Stack

- **Frontend:** React.js / Vite (TypeScript)
- **Database:** Neon Console (Serverless PostgreSQL)
- **Package Manager:** pnpm
